# ADE Insight

ADE Insight is a professional test results analyser for BS EN 22041.

It ingests:
- Temperature / RH CSV logs
- Electrical power analyser TXT logs

And produces:
- Aligned datasets
- QC validation
- Summary tables
- Electrical metrics
- Temperature statistics
- Report-ready PNG plots
